   ```
   npm install
   ```
3. **Install client dependencies**
   ```
   cd client
   npm install
   ```
4. **Configure environment variables**
   - Create a `.env` file in the root and add your MongoDB URI and JWT secret.

5. **Start the server**
   ```
   npm start
   ```
6. **Start the client**
   ```
   cd client
   npm start
   ```
